# <span style="color:red;">Now-Playing-RED</span>

*A <span style="color:red;">BLOOD RED</span> - NowPlaying Widget With Media Controls and Slider!*

---

## INSTALL

Use the [Makefile](./Makefile).

+ `$	make install`

## UNINSTALL

Use the [Makefile](./Makefile).

+ `$ make remove`
